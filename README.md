Simple ffmpeginput plugin for OBS (http://obsproject.com)

This piece of code created during researching of OBSApi architecture,
so don't take anything here seriously. ^_^

Dependencies:
    1) FFMpeg libraries:
        * libavformat
        * libavcodec
        * libavresample
        * libswscale
        * libavutil
    2) Boost libraries:
        * libboost_chrono
        * libboost_date_time
        * libboost_system
        * libboost_thread

Supported OBS version: 0.473b
